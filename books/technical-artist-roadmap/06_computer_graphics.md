# ポリゴン・点群

コンピューターグラフィックスにおいて、形状を表示させるための単位

この三角形の複合で様々な形状を表現できる。

１ポリゴンに３つの頂点（vertices）がある。またそれぞれのvertexには順番indicesがあり、その順番によって、ポリゴンの裏と表を表現する。


また、点で形状を表現する点群という方法もある。

面の情報は持たないが、データとしてはポリゴンより扱いやすくは有る。

# プロジェクション

三次元空間のパラメータを2次元空間（ピクセル）に投影する方法

ビュー行列とプロジェクション行列を使って、カメラのシュミレートをする

ビュー行列とは、カメラの位置・回転を表した4x4の行列（実際にはカメラの姿勢を表した行列の逆行列となる）。モデル行列とビュー行列をかけることによって、カメラの位置を原点とした座標系になる。

プロジェクション行列は、カメラの錐台を表した行列、

錐台を行列として表現したものを行列として計算に使うことによって、楽に結果を出せる

# ライティング・シェーディング

シェーディングとは、ポリゴンの色を決定するための陰影を表現する手法です。

一番シンプルなLambert反射の説明から入って、

物理ベースレンダリング （PBR, BRDFモデル）の説明までできるとOK

Diffuse, Specular, 

Shadow


影は光源というカメラから見た見えない領域の投影

**レイトレーシング**

カメラに入ってくる光の挙動をトレース（追跡）することによってレンダリングする手法。

パストレーシングは１ピクセル内でランダムにたくさんレイトレをして結果を重ねていく方法

Z-Buffer法

ポリゴンやピクセル単位で奥行きでソートして、順番に描画していく手法

**フォーワードレンダリング**

ポリゴン・ピクセルごとにライティングが計算されて結果が表示される。

ポリゴンの表面にシェーダーが乗っかってるイメージ

**ディファードレンダリング**

レンダリングに必要なパラメーターをいくつかのGバッファー（画像）を作って保持しておいて、あとで画像合成のノリで合成して結果を作る手法。

# 色空間とカラープロファイル

無限色ある自然界の色を、数値的に表現するための仕様

同じRGBでも、sRGBとAdobeRGBでは結果が違う。

現在映像業界ではACESフォーマットが主流になりつつある。
が表現力が豊かな分ハイエンド寄りかつ、データも重くなる。

ガンマ補正

ディスプレイとコンピューターとでは陰影の表現がズレている。のを補正する。


**LUT**

色の変換ロジックを保存した画像みたいなデータのこと。

カラーグレーディングは、本来たくさんのパラメータをつかった計算が必要だが、LUTでは常にその結果のみを参照するので、高速にカラーグレーディングができる。

たとえば色のIndexは0~255が3つあるので、256x256x256三次元テクスチャがあれば、すべての色の対応関係が表現できることになる。
